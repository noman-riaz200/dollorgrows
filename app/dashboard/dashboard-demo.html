Responsive Breakpoints

The dashboard uses CSS media queries to adapt to different screen sizes:

Desktop (1024px+)
3-column grid, full charts
Tablet (768px-1024px)
2-column grid, stacked charts
Mobile (480px-768px)
1-column, simplified layout
